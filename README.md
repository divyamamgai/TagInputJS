# TagInputJS
A jQuery plugin to create a simple Tag input element for your HTML forms.

As simple as,
```HTML
<input id="Keywords" name="Keywords"
        type="text"
        data-TagInput
        data-Glue=","
        data-MinLength="1"
        data-MaxLength="16"
        data-LabelAnimationClass="Material"
        value="HTML,CSS">
```

**Documentation** : https://divyamamgai.github.io/TagInputJS/
